# Flipkart-Sales-Analysis-Dashboard
## Project Description

This project presents an interactive Power BI dashboard built using Flipkart e-commerce data to analyze sales performance, customer behavior, and product trends.

The dashboard provides a comprehensive view of key business metrics such as revenue, order volume, product categories, customer segmentation, and regional performance. It is designed to help stakeholders make data-driven decisions by identifying high-performing products, understanding customer preferences, and tracking sales growth over time.

## Key Features
- Sales trend analysis (daily/monthly/yearly)
- Product category performance comparison
- Region-wise sales distribution
- Customer segmentation insights
- Revenue and profit analysis
- Ratings and reviews analysis (if included)
- Tools & Technologies
# Power BI (.pbix)
Data cleaning & transformation using Power Query
Data modeling with relationships and DAX measures
